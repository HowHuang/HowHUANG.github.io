This is a Centos7 machine

1 Install icc ifort

It is quite easy for a root user

I chose Intel® oneAPI HPC Toolkit (https://www.intel.com/content/www/us/en/developer/tools/oneapi/hpc-toolkit.html#gs.bec8rv)

Then just follow the installation instructions

But now if you run which ifort, you can not find the compiler

Add it to profile will load compiler duing every login, so I add a reminder in /etc/profile
